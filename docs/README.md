# Natural Gas Storage Forecast

A resume-level data science project forecasting weekly U.S. Lower 48 natural gas storage changes.

Current components:

- EIA storage data ingestion and cleaning
- Open-Meteo weather ingestion
- Seasonal baseline forecasting
- Fourier regression
- SARIMA comparison
- Residual diagnostics
- Population-weighted HDD/CDD features
